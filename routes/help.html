Grids

Using the Grids

  • Click on grid row to select the row
  • Re-arrange column order by dragging and dropping column headers
  • Column width may be changed by dragiing edges of column headers
  • Page navigation using status row page controls
    • Status display on bottom of grid will display total number of row entries and which ones are currently visible
    • Use the Goto Row, Show Rows, Page Previous, Page Next buttons to go to a specific page number, change number of rows visible at one time and page respectively
  • Data filtering
    • Select the colunn to filter from the dropdown list
    • Enter string to search for and press ENTER
  • Advanced data filtering
    • Click on advanced
    • Click on icon
    • Select the type of filter and enter filter data
    • Example, select filter contains and enter Cafe to filter grid for only rows where selected filter column contains the word Cafe and press ENTER
  • Sorting data
    • Click on the column title to sort by that column - each click will toggle between sorting by ascending and descending order

Orders

Add/Modify

  • Enter an optional P.O. (Purchase Order) number from the client
  • Select the client the order belongs to
  • Enter an optional name of thr order - by default it will be the client's name
  • Select the version of the order you wish to work on if there are multiple versions - note this does will not make the selected version the active one unless you Submit (save) the order
  • Select an optional start date to begin processing the order
  • Select an optional required by date for the order
  • Click on Submit to save a new order or the selected order
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected order
  • Click on Duplicate to make a copy of the selected order - the name of the order will have the text "copy" appended to it's name and the next available order number will be used
  • Click on New Version to add a new version of the order's details based on the currently selected order version (which does not need to be the current active version)
  • Click on Search to display the order search popup
  • Click on Invoice to convert this order to an invoice

Invoice To

First select an order from the grid of orders
  • By default invoice to name and address details are populated from the client's primary contact details
  • Enter an optional alternative name and address to override

Ship To

First select an order from the grid of orders
  • By default ship to to name and address details are populated from the client's primary contact details
  • Enter an optional alternative name and address to override

Statuses

First select an order from the grid of orders
  • Select the new status for the selected order from the dropdown list - you may change the order status as often as you like - orders will be displayed with the latest status selected from here
  • Enter an optional carrier or courier name
  • Enter an optional consignment note number (tracking number)

Details

First select an order from the grid of orders
  • Note changes here are applied to the currently selected order version even if that version is not the currently active version
  • Select a product from the dropdown list to add to the selected order
    • if a price entry is available for the selected product it will be populated automatically
    • if more than one price is available, the list of available prices will be displayed for selection
    • Click on Magnifying Glass icon to lookup the price(s) of the selected product
  • Enter the quantity required for the selected product

Notes

First select an order from the grid of orders
  • Enter notes against the selected order - this is handy for tracking conversations with clients
;

Attachments

First select an order from the grid of orders
  • Enter an optional description for the attachment you wish to upload
  • Click to browse for attachments, or drag and drop your attachment (file, document or image)
  • Click on Submit to save description of the selected attachment
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected attachments

Search

  • Enter your search criteria - the string you search for does not need to be complete - partial strings will also be searched
  • Click on Search to start the search - results will be displayed in the orders grid
  • Click on Reset to reset the list of orders

 

 

Product Categories

Add/Modify

  • Select an optional parent product category if the current or new category is a subcategory
  • Enter the name of the category
  • Enter an optional code for the category
  • Enter an optional barcode for the category
  • Enter optional notes for the category
  • Click on Submit to save a new category or the selected category
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected category - Note that subcategories will be moved to the same level or the parent of this category

Chart

  • Displays a visual chart representing the hierarchy and relationship between product categories

 

 

Products

Add/Modify

  • Select an optional product category for the new or selected product
  • Enter the name of the product - note you may have a duplicate name product under a different category
  • Enter an optional product code
  • Enter an optional barcode for the product - entering TAB will display the barcode if a barcode type is selected
  • Enter the cost price of the product - entering TAB will calculate the GST
  • Enter an optional UOM (Unit Of Measure) - e.g. kg, ml, lb
  • Enter an optional UOM size (e.g. 5kg)
  • Enter an optional minimum stock quantity
  • Enter stock quantity at which an alert will be generated
  • Enter an optioal product dimension and weight
  • Enter optional notes for this product
  • Click on Submit to save a new product or the selected product
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected product

Pricing

First select a product from the grid of products
  • Select an optional client for the selected product if the price applies specifically for this client
  • Enter a sell price for the selected product - entering TAB will calculate the GST
  • Enter an optional minimum and maximum quantity if this price applies for a specific volume quantity
  • Enter optional notes for this price entry
  • Click on Submit to save a new price entry or the selected price entrys
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected price entry

Attributes

First select a product from the grid of products
  • Enter optional custom attributes for the selected product

 

 

Product Templates

Add/Modify

  • Select an optional parent template for the new or selected template
  • Select an optional client if this template is only applicable to the selected client
  • Enter the name of the template
  • Enter an optional default sell price - entering TAB will calculate the GST
  • Enter the quantity of inventory stock this template will produce
  • Enter optional notes for this template
  • Click on Submit to save a new product template or the selected product template
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected product template - Note that all subtemplates will be removed

Details

First select a template from the grid of templates
  • Select a product from the dropdown list to add to the selected template
    • if a price entry is available for the selected product it will be populated automatically
    • if more than one price is available, the list of available prices will be displayed for selection
  • Enter the quantity required for the selected product
  • Select Cascade to apply changes to all subtemplates with the same product

 

 

Clients

Add/Modify

  • Select an optional parent client if the current or new client is a subclient (e.g. branch or division)
  • Enter the name of the client
  • Enter an optional client code
  • Enter the name of the primary client contact
  • Enter the email address of the primary contact
  • Enter the phone or mobile number of the primary contact
  • Enter the FAX number of the primary contact
  • Enter the address of the client
  • Select a default invoice template from the dropdown list
  • Select Supplier if this client is also a supplier
  • Enter optional notes
  • Click on Submit to save a new client or the selected client
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected client

Shipping

First select a client from the grid of clients
  • Enter an optional shipping address for the selected client
  • Click on Submit to save shipping details for the selected client
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected client

Banking

First select a client from the grid of clients
  • Enter the name of the bank
  • Enter the bank's BSB number
  • Enter the bank account number
  • Enter the optional bank account name
  • Click on Submit to save banking details for the selected client
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected client

Limits

First select a client from the grid of clients
  • Enter the payment terms for the selected client (in days - usually 7, 14, 30, 60, 90)
  • Enter an optional line limit - the maximum amount for a single order line item
  • Enter an potional order limit - the maximum amount for a single order
  • Enter an optional credit limit - the maximum amount for all oustanding orders
  • Click on Submit to save limit details for the selected client
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected client

 

 

Locations

Add/Modify

Locations can be used to group stock in inventory - e.g. warehouses, trucks etc
  • Select an optional parent location if the current or new location is a sublocation (e.g. floor, room of a building)
  • Enter the name of the location
  • Enter an optional location code
  • Enter an optional GPS coordinate for this location
  • Enter the address of the location - the GPS coordinate of this location will be populated automatically if available
  • Enter optional notes
  • Click on Submit to save a new location or the selected location
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected location

Attributes

First select a location from the grid of location
  • Enter optional custom attributes for the selected location

 

 

Users

Add/Modify

  • Enter the login user name - must be unique
  • Enter the user's real name
  • Enter an optional email address
  • Enter an optional phone or mobile number
  • Select Administrator to grant user full administrative rights and access
  • Select an optional avatar (icon) to represent the user
  • Enter a password - leave this blank to keep existing password if modifying an existing user
  • Re-enter password to confirm
  • Enter optional notes
  • Click on Submit to save a new user or the selected user
  • Click on Clear to reset all fields
  • Click on Delete to remove the selected user

Permissions

First select a user from the grid of users
  • For non-administrators, you must select (enable) access to the listed rights
  • User will be required to logout and login again for the changes to take affect
  • These settings have no affect for administrators

 

 

Config

Modify system wide preferences

  • Enter your starting invoice and order numbering scheme
    • Numbers can start with a non-numeric e.g. INV-0001
    • The numeric end will be incremented automatically for each new invoice/order
    • Note the displayed number is the next available number
  • Select Quote First to create new orders with a status of quote, otherwise new orders will be created with a status of order
  • Click on Submit to save your config changes
  • Click on Clear to reset all fields
